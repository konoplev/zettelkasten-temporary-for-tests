# Бином Ньютона

Формула Бинома Ньютона:

$$(a+b)^n=\sum_{k=0}^n\binom{n}{k}a^{n-k}b^k$$

Формула Бинома Ньютона используется для расчета разложения бинома в степени $n$, где $a$ и $b$ - любые числа или переменные, а $\binom{n}{k}$ - биномиальный коэффициент.

Биномиальный коэффициент $\binom{n}{k}$, где $n$ и $k$ - целые неотрицательные числа, представляет собой число способов выбрать подмножество из $k$ элементов из множества, содержащего $n$ элементов, при условии, что порядок элементов в подмножестве не имеет значения. Математически это выражается как:

$$\binom{n}{k}=\frac{n!}{k!(n-k)!}$$

где $n!$ - факториал числа $n$, равный произведению всех положительных целых чисел от $1$ до $n$.

Существует несколько способов получения формулы биномиального коэффициента. Один из наиболее распространенных способов - это комбинаторный подход, основанный на идее подсчета количества способов выбрать подмножество из $k$ элементов из множества, содержащего $n$ элементов.
