# Наибольший общий делитель

Наибольший общий делитель (НОД) двух или более чисел - это наибольшее число, которое делит каждое из этих чисел без остатка. Существует несколько способов нахождения НОД, включая:

1.  Метод Эвклида:

-   Раздели большее число на меньшее.
-   Если делится без остатка, то меньшее число и есть НОД.
-   Если есть остаток, то замени большее число на остаток от деления и повтори операцию с меньшим числом и остатком.
-   Продолжай повторять шаги до тех пор, пока не получишь два числа, которые делятся нацело - это и есть НОД.

2.  Разложение чисел на простые множители:

-   Разложи каждое число на простые множители.
-   Найди все общие простые множители у этих чисел.
-   Умножь все найденные общие множители.

Например, для нахождения НОД чисел 12 и 18:

1.  Метод Эвклида:

-   18 / 12 = 1 ост. 6
-   12 / 6 = 2 ост. 0
-   НОД(12, 18) = 6

2.  Разложение на простые множители:

-   12 = 2 * 2 * 3
-   18 = 2 * 3 * 3
-   Общие множители: 2 и 3
-   НОД(12, 18) = 2 * 3 = 6

Оба метода дают одинаковый результат, но второй метод может быть удобен для нахождения НОД большего числа.
