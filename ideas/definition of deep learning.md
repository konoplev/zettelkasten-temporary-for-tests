# definition of deep learning
Deep learning is a class of machine learning algorithm that uses multiple stacked layers of processing units to learn high-level representations from unstructured data.

Instead of using tabular data (where columns are features), a deep learning model can learn how to build high-level informative features by itself, directly from the unstructured data (for example images or texts).
