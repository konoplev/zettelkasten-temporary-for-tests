# Метод математической индукции

Чтобы доказать, что некоторая теорема верна для всякого натурального числа `n`, достаточно доказать
1. эта теорема справедлива для n=1
2. что если теорема справедлива для некоторого n, то она верна и для следующего натурального числа n+1

### Пример 1

Доказать, что $1 + 2 + ... + n = \frac{n\times(n+1)}{2}$

Доказательство. Это очевидно правда для 1. 

Если это справедливо для n, то и для n+1, получаем

$1 + 2 + ... + n + (n + 1)= \frac{(n+1)\times(n+2)}{2}$

Заменим $1 + 2 + ... + n$ правой частью из первого утверждения. Получается

$\frac{n\times(n+1)}{2} + (n + 1)= \frac{(n+1)\times(n+2)}{2}$

Сократим (n + 1)

$\frac{n}{2} + 1= \frac{(n+2)}{2}$

и очевидно это справедливо

$\frac{n}{2} + 1= \frac{n}{2} + 1$

### Пример 2

Доказать, что $1^2 + 2^2 + ... + n^2 = \frac{n(n+1)(2n+1)}{6}$

При n=1 очевидно выполняется. Если справедливо при n, то при n+1: $1^2 + 2^2 + ... + n^2 + (n+1)^2 =   \frac{(n+1)(n+1+1)(2(n+1) + 1)}{6}$

подставим в левую часть равенства правую часть из равенства при n

$\frac{n(n+1)(2n+1)}{6} + (n+1)^2 = \frac{(n+1)(n+1+1)(2(n+1) + 1)}{6}$

можно раскрыть скобки и доказать, что это справедливо
