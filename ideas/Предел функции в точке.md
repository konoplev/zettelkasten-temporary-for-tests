# Предел функции в точке

Предел функции f(x) в точке x=a - это значение, к которому стремится f(x), когда x приближается к a (но не равняется ему). Формально, можно записать это так:

$lim_{x->a} f(x) = L$

где L - число (если он существует) и означает, что для любого положительного числа ε (epsilon) существует положительное число δ (delta), такое что если 0 < |x - a| < δ, то |f(x) - L| < ε.

В этом определении, ε - это сколь угодно маленькое положительное число, а δ - это соответствующее положительное число, которое показывает, насколько близко нужно быть к a, чтобы значения f(x) были на расстоянии менее ε от L.

### Критерий Коши

Критерий Коши - это один из способов определения существования предела функции в точке a. Он гласит, что предел функции f(x) в точке x=a существует тогда и только тогда, когда для любого положительного числа ε (epsilon) существует положительное число δ (delta), такое что если 0 < |x - a| < δ и 0 < |y - a| < δ, то |f(x) - f(y)| < ε.

Другими словами, если для любой пары точек x и y в окрестности точки a, расстояние между значениями функции f(x) и f(y) меньше, чем ε, то можно сделать вывод, что предел функции f(x) существует в точке a.

Критерий Коши - это более строгий подход, чем просто требование, что функция должна стремиться к одному и тому же значению при приближении x к a с двух сторон. Он позволяет более точно определить сходимость функции в точке и устанавливает условие, что значения функции должны быть сколь угодно близки в любой точке, близкой к точке a.

### Односторонний предел
Односторонний предел - это предел функции f(x) в точке x=a, когда x приближается к a только с одной стороны, либо справа, либо слева от a. Формально, можно записать это так:

-   Односторонний предел функции f(x) в точке x=a справа равен L, если для любого положительного числа ε (epsilon) существует положительное число δ (delta), такое что если a < x < a + δ, то |f(x) - L| < ε.
-   Односторонний предел функции f(x) в точке x=a слева равен M, если для любого положительного числа ε (epsilon) существует положительное число δ (delta), такое что если a - δ < x < a, то |f(x) - M| < ε.

Если односторонние пределы справа и слева существуют, но не равны друг другу, то предел функции f(x) в точке x=a не существует.

Односторонний предел позволяет изучать поведение функции в точке на каждой стороне от нее, и может быть использован, например, для определения наличия разрывов в функции или для изучения ее поведения на границе определения.

### Бесконечный предел

Бесконечный предел - это предел функции f(x), который не сходится к конкретному числу, а стремится к бесконечности. Математически, можно записать это так:

-   Предел функции f(x) равен бесконечности, если для любого положительного числа M (большого числа) существует положительное число δ (delta), такое что если |x - a| < δ, то f(x) > M (f(x) растет без ограничения).
    
-   Предел функции f(x) равен минус бесконечности, если для любого отрицательного числа M (малого числа) существует положительное число δ (delta), такое что если |x - a| < δ, то f(x) < M (f(x) убывает без ограничения).
    

Если функция f(x) стремится к бесконечности справа или слева от точки a, то используются односторонние пределы.

Бесконечный предел возникает, когда значения функции f(x) становятся все больше и больше или все меньше и меньше по мере приближения аргумента x к точке a. Такой предел может быть использован, например, для описания скорости роста или убывания функции, которая не имеет конечного предела.
