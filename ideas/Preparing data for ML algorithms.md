# Preparing data for ML algorithms
Important part is analysing the data and choosing best algorithm. 

For transformations can be used [[NumPy quickly]] , `pandas` and `matplotlib`

To visualize and analyse the `seaborn` can be used.

Sometimes an algorithm (for example a liner regression) can require normally distributed data. It's not always the case, but log-normal distributed can be transformed to normally distributed by using [[Log-transformation]].
