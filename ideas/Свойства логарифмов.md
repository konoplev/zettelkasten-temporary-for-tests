# Свойства логарифмов
see [[What is Logarithm?]]

Для начала, определим логарифм с основанием $a$ от числа $x$ как:

$$\log_{a} x$$
если $a^b = x$, то логарифм с основанием $a$ от $x$ равен $b$:

$$\log_{a} x = b$$

Теперь перейдем к свойствам логарифмов.

1.  Свойство умножения:

$$\log_{a}(xy) = \log_{a} x + \log_{a} y$$

Следует из $$a^x \times a^y = a^{x + y}$$

2.  Свойство деления:

$$\log_{a}\frac{x}{y} = \log_{a} x - \log_{a} y$$

3.  Свойство возведения в степень:

$$\log_{a}(x^{n}) = n\log_{a} x$$

4.  Свойство изменения основания:

$$\log_{a} x = \frac{\log_{b} x}{\log_{b} a}$$

Здесь $x, y$ - положительные числа, $a, b$ - положительные числа, не равные 1, а $n$ - любое число.
