# Функция

Функция - это математический объект, который связывает каждый элемент из одного множества (называемого областью определения) с единственным элементом из другого множества (называемого областью значений). Обозначение функции часто записывается как $f(x)$, где $x$ - элемент области определения, а $f(x)$ - соответствующий ему элемент области значений. Функции могут быть заданы явно (например, $f(x) = x^2$) или неявно (например, как решение уравнения $x^2 + y^2 = 1$ относительно $y$).

### Область опеределения
Область определения может быть произвольным множеством, в зависимости от того, какие значения аргумента функции принимаются и какие допустимы для данной функции. Например, для функции $f(x) = \sqrt{x}$ область определения - это множество всех неотрицательных чисел, тогда как для функции $g(x) = \frac{1}{x}$ область определения - это множество всех действительных чисел, кроме нуля.

### Многозначная функция
Многозначная функция - это функция, которая сопоставляет одному элементу из области определения несколько элементов из области значений. Многозначные функции часто встречаются в комплексном анализе и теории функций.

Например, корневая многозначная функция $\sqrt{x}$ сопоставляет каждому положительному числу два значения: $\sqrt{x}$ и $-\sqrt{x}$. Еще один пример многозначной функции - арктангенс, обозначаемый как $\operatorname{atan}(x)$ или $\operatorname{arctan}(x)$. Арктангенс определен как функция, обратная к тангенсу, и может принимать несколько значений для каждого значения аргумента. Например, $\operatorname{atan}(1) = \frac{\pi}{4}$, но также можно показать, что $\operatorname{atan}(1) = \frac{5\pi}{4}$ и $\operatorname{atan}(1) = \frac{9\pi}{4}$ и т.д.

Еще один пример многозначной функции - многозначный логарифм. Многозначный логарифм определяется как решение уравнения $e^z = w$ для заданного комплексного числа $w$. Каждому значению $w$ может соответствовать бесконечное число значений многозначного логарифма.

### Обратная функция

Обратной функцией к функции $f$ называется такая функция $g$, что выполнены два условия:

1.  Область определения функции $g$ равна области значений функции $f$, то есть $D_g = R_f$.
2.  Для любого $x$ из области значений функции $f$ выполнено равенство $g(f(x))=x$.

Другими словами, если $f(x)$ является функцией одного аргумента и для каждого значения $x$ она дает какое-то значение $y=f(x)$, то обратная функция $g(y)$ дает нам обратно значение $x=g(f(x))$.

Обратная функция может существовать не для всех функций, а только для тех, которые обладают свойством инъективности (то есть для каждого $y$ из области значений функции $f$ существует не более одного $x$ из области определения функции $f$ такого, что $f(x) = y$).

### Пример 1

Определить область существования функции $\sqrt{3x - x^3}$
Для того чтобы определить область существования функции $\sqrt{3x - x^3}$, необходимо решить неравенство $3x - x^3 \geq 0$, так как в знаменателе не может быть отрицательного числа.


Можно факторизовать левую часть неравенства: $3x - x^3 = x(3-x)(1+x)$. Таким образом, решая неравенство $x(3-x)(1+x) \geq 0$, мы найдем область существования функции.​​
