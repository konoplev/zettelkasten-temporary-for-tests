# Признаки существования [[предел]]а

1. Если $y_n \le x_n \le z_n$, и $\lim_{n \to \infty}y_n = \lim_{n \to \infty}z_n = c$, то $\lim_{n \to \infty}x_n = c$ 
2. Монотонная ($x_n < x_{n+1}$) и ограниченная ($\exists m \in R: x_n \le m, \forall n \in N$) последовательность имеет предел
3. Критерий Коши: Пусть $x_n$ — числовая последовательность (последовательность с элементами из R). $x_n$ имеет предел в R тогда и только тогда, когда: для $\forall \varepsilon, \exists N \in \mathbf{N}$ такое что $\forall m,n > N: |a_m - a_n| < \varepsilon$, иными словами, предел существует тогда и только тогда, когда для любого числа, пусть даже очень маленького, мы можем найти индекс, начиная с которого разница между любыми элементами последовательности не превышает нашего числа.

### Пример 1
Утверждение $\lim_{n \to \infty}x_n=\infty$ можно сформулировать с помощью неравенств следующим образом:

Для любого положительного числа $M$ найдется номер $N$, начиная с которого все члены последовательности $x_n$ будут больше $M$, т.е.
$\forall M>0, \exists N : n \ge N => x_n > M$
Это утверждение означает, что последовательность $x_n$ растет неограниченно и может принимать сколь угодно большие значения при достаточно больших номерах.

### Пример 2
Показать, что $x_n = n^{(-1)^n}$ не ограничена, однако не является бесконечно большой при $n \to \infty$

$x_n=n^{-1}$, а когда $n$ - нечетное, то $(-1)^n=-1$, и $x_n=n$. Следовательно, последовательность $x_n$ чередует возрастание и убывание, и каждый ее член больше предыдущего или меньше предыдущего в зависимости от того, является ли $n$ четным или нечетным. Таким образом, последовательность $x_n$ не ограничена.

Теперь покажем, что $x_n$ не является бесконечно большой при $n\to\infty$. Рассмотрим два подслучая:

-   Если $n$ - четное, то $x_n=n^{-1} \to 0$ при $n \to \infty$.
-   Если $n$ - нечетное, то $x_n=n \to \infty$ при $n \to \infty$.

Таким образом, мы доказали, что последовательность $x_n$ не является ограниченной, но также не является бесконечно большой при $n\to\infty$.


#### Пример 3
Пользуясь критерием Коши, доказать сходимость следующей последовательности: 

$x_n = a_0 + a_1\times q + ... + a_n \times q^n$

где $|a_k| < M (k = 0,1,.. )$ и $|q|<1$

Для доказательства сходимости последовательности $x_n$ с помощью критерия Коши необходимо показать, что для любого положительного числа $\epsilon$ существует такой номер $N$, начиная с которого все пары элементов последовательности $x_n$ различаются не более чем на $\epsilon$. То есть:

$$\forall \epsilon > 0 \quad \exists N \quad \forall n, m > N \quad |x_n - x_m| < \epsilon$$

Разность $|x_n - x_m|$ можно оценить следующим образом:

$$|x_n - x_m| = |(a_0 + a_1 q + ... + a_n q^n) - (a_0 + a_1 q + ... + a_m q^m)|$$

Заметим, что суммы $a_0 + a_1 q + ... + a_n q^n$ и $a_0 + a_1 q + ... + a_m q^m$ совпадают до $m$-го члена, поэтому:

$$|x_n - x_m| = |a_{m+1} q^{m+1} + ... + a_n q^n| \leq |a_{m+1}q^{m+1}| + ... + |a_nq^n|$$

Воспользуемся теперь условием $|a_k| < M$ и $|q|<1$ для оценки правой части:

$$|a_{m+1}q^{m+1}| + ... + |a_nq^n| < Mq^{m+1} + ... + Mq^n = Mq^{m+1}(1 + q + ... + q^{n-m-1})$$

Сумма геометрической прогрессии $1 + q + ... + q^{n-m-1}$ равна $\frac{q^{n-m}-1}{q-1}$, поэтому:

$$Mq^{m+1}(1 + q + ... + q^{n-m-1}) = Mq^{m+1}\frac{q^{n-m}-1}{q-1} = \frac{M}{1-q}(q^{n+1}-q^{m+1})$$

Таким образом, получаем:

$$|x_n - x_m| \leq \frac{M}{1-q}(q^{n+1}-q^{m+1})$$

Чтобы выполнялось условие $|x_n - x_m| < \epsilon$, необходимо, чтобы выполнялось неравенство:

$$\frac{M}{1-q}(q^{n+1}-q^{m+1}) < \epsilon$$

Решим его относительно $n$:

$$q^{n+1} < \frac{\epsilon(1-q)}{M} + q^{m+1}$$

Так как $|q|<1$, то $q^{m+1}$ может быть оценено как любая константа, не зависящая от $n$. Таким образом, мы можем выбрать номер $N$ так, чтобы для всех $n > m$

$$q^{n+1} < \frac{\epsilon(1-q)}{M} + q^{m+1} \leq \frac{\epsilon(1-q)}{M} + K$$

где $K$ - некоторая константа. Тогда мы получаем:

$$n+1 < \log_q\left(\frac{\epsilon(1-q)}{M} + K\right)$$

Таким образом, для всех $n > m > N = \lceil\log_q(\frac{\epsilon(1-q)}{M} + K)\rceil - 1$ выполняется условие $|x_n - x_m| < \epsilon$, что и доказывает сходимость последовательности $x_n$ по критерию Коши.

Итак, мы показали, что для любого положительного числа $\epsilon$ существует такой номер $N$, начиная с которого все пары элементов последовательности $x_n$ различаются не более чем на $\epsilon$, что и означает, что последовательность $x_n$ сходится.
